# Desafio: Explorando Colaboração e Markdown

Este repositório faz parte do desafio prático da **DIO**, focado em aprender a utilizar Git/GitHub e a linguagem de marcação Markdown para documentação de projetos.

## 📝 O que aprendi
- [x] Criação de repositórios no GitHub.
- [x] Versionamento de código com Git.
- [x] Documentação técnica utilizando Markdown.
- [x] Colaboração em projetos open source.

## 🛠️ Tecnologias Utilizadas
- Git
- GitHub
- Markdown
